# V2 Asset Batch 02

Batch 02は高品質生成アセットへ置換するためのプロンプト仕様である。この作業では外部素材をダウンロードせず、既存PNGを`approved`扱いにしない。

共通方針：

- 完全オリジナルの暗い結晶迷宮ファンタジー。
- 画像内に文字、ロゴ、透かしを入れない。
- 小サイズで読めるシルエットを優先する。
- 透過が必要なアイコンは透明背景。
- 人間レビュー前は`prompt_ready`または`generated_unreviewed`にする。

対象はタイトルキービジュアル、初期キャラクター、初期武器、初期パッシブ、5分ボス、星屑平原、ショップカテゴリ、進化通知、ラッシュ表示、宝箱、目的地アイコンである。
