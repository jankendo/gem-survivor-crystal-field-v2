# Item Placement Contract

対象はEXPジェム、フィールドドロップ、復活ドロップ、宝箱、フィールド武器、フィールドパッシブ、イベント報酬、ギミック報酬、結晶破壊報酬、将来のpickupである。

最終配置位置は次を満たす。

- マップ範囲内で端に近すぎない。
- 歩行可能床上で、半径と余白を含めて壁へ食い込まない。
- プレイヤー開始地点と同じ到達可能連結成分に属する。
- 有効な部屋または通路に属する。
- 同一seedで同じ配置になる。

生成元は候補位置を出し、`ItemPlacementSystem.resolve_valid_pickup_position()`へ渡す。無効な場合は安全セルプールから決定的に再抽選し、安全セルがない場合は生成を中止する。
