# iOS Phase 4 Real Device Checklist

This checklist is pending until a physical iPhone/iPad is available.

## Devices

- iPhone SE or smallest supported landscape iPhone
- iPhone with notch / Dynamic Island
- iPhone Pro Max class
- iPad 11-inch class
- iPad 12.9-inch class

## Checks

- Title start button visible without scrolling.
- All title navigation buttons reachable by vertical scroll.
- No title control under notch, rounded corner, or home indicator.
- Landscape-left and landscape-right both valid.
- Touch targets feel tappable without accidental adjacent taps.
- Game screen HUD remains inside Safe Play Area.
- Environment art does not hide EXP gems, field drops, enemies, boss warnings, or player.
- 10-minute iOS low-profile run has no visible hitching or thermal warning.
