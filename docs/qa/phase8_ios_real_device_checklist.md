# Phase 8 iOS実機チェックリスト

## 前提

GitHub Actionsの未署名IPA、headless、Simulatorは実iPhone性能の証明に使わない。署名可能な実機ビルドをiPhoneへ導入し、同一seed 60606で`ios_low`と`ios_minimal`を比較する。

## 操作

- [ ] Safe Area内で設定リストと全OptionButtonを操作できる
- [ ] 設定変更後もスクロール位置とフォーカスが維持される
- [ ] 倍速を0.9秒長押しで固定し、短いタップで解除できる
- [ ] ポーズ、マップ、レベルアップ、宝箱中は1倍となり、閉じると固定へ戻る
- [ ] 手動終了後にリザルトが一度だけ表示され、タイトルへ戻れる
- [ ] 武器コアとパッシブコアをフィールドとミニマップで識別できる
- [ ] 解放前の対象が画面、ミニマップ、矢印に出ない
- [ ] 各イベント矢印が実際の対象へ向き、対象消滅後に消える
- [ ] ショップ条件と不足値が横画面で読める

## 性能

- [ ] 30分プレイでフリーズ、100ms超過、入力欠落がない
- [ ] `ios_minimal`で背景粒子、ダメージ数字、弾軌跡が表示されない
- [ ] Criticalのボス予告、危険予告、取得結果が欠落しない
- [ ] 敵数、弾、撃破、EXP、報酬が`ios_low`と一致する
- [ ] InstrumentsでCPU/GPU frame time、hitch、allocationを採取する
- [ ] `ProcessInfo.thermalState`の推移と端末表面温度を記録する
- [ ] 30分のbattery減少率を同一端末・同一輝度で比較する

## 記録

端末、iOS、ビルドSHA、開始温度、終了温度、battery開始/終了、p50/p95/p99、16.67/33/100ms超過、スクリーンショットを添付する。
