# v2 Gameplay Pillars

## 1. 探索型サバイバーを維持する

フィールド探索、危険地帯、報酬部屋、ギミック、ドロップ復活、フィールド装備の強みを残す。v2では探索の意味をさらに見えやすくし、どこへ向かうと得なのかをHUDと通知で補助する。

## 2. ビルドが育つ感覚を強める

武器、パッシブ、進化、過充電、シナジーは現行の厚みを維持する。プレイヤーには「今の主力」「進化候補」「相性が発動した理由」を短く見せる。

## 3. ラン内の山を増やす

レベルアップ、進化、全ジェム回収、ボス出現/撃破、連続撃破、イベント成功を、分かりやすい通知と短時間のMomentumでまとめる。新しい数値効果は控えめにし、まずはスコア倍率と可視化から始める。

## 4. 失敗しても次が分かる

リザルトは単なる数値一覧ではなく、今回のビルド概要、強かった武器、伸びた進捗、次に狙う要素を示す。初心者には「序盤は敵を倒しやすい武器」「通貨を強化へ使う」などの実用的な次導線を返す。

## 5. 低リスクで積み上げる

v2の新機能はラン内状態を基本にする。保存データを変える必要がある機能は後回しにし、既存テストとスモークを壊さない範囲で拡張する。
# Phase 3 Pillar Clarification

The core value is exploration, crystal mining, risky rewards, and build completion. New content count is secondary to readability, meaningful choices, and a reliable first-run loop.
