# Japanese Style Guide

プレイヤー向け表示は日本語を正本とする。内部ID、ファイル名、bundle id、exe名は英語のままでよいが、UIへ直接出さない。

- Momentum：ラッシュ
- Momentum Tier：ラッシュ段階
- Level Up：レベルアップ
- Result：リザルト
- Locked：未解放
- Unlocked：解放済み
- Purchased：購入済み
- Available：購入可能
- Requirement：購入条件
- Passive：パッシブ
- Evolution：進化
- Overclock：過充電
- Currency：クリスタル貨
- READY：使用可能

文体は短く、直訳調を避ける。「購入条件を満たしていません」「購入すると、以後のランで候補に出現します。」を基準表現にする。
