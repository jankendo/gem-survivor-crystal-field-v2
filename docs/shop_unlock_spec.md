# Shop Unlock Spec

永久使用可能の正式条件は「スターター」または「ショップ購入済み」のどちらかだけである。

条件達成、実績、クエスト、リザルト、ボス撃破、生存時間、図鑑発見は、永久解放を直接発生させない。これらは商品公開、購入条件達成、購入おすすめ、通貨報酬、図鑑説明公開へ変換する。

`ShopEntitlementSystem`がスターター判定、商品公開判定、購入条件判定、購入済み判定、使用可能判定、候補プール制御、セーブ移行を一元管理する。

旧セーブでは、購入証跡のある商品とスターターを保持する。条件解放だけの非スターターは再ロックし、`shop_available`へ移して初回だけ通知する。
